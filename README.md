# TianWen

## 如何使用?
  将文件解压放入 /Starbound/win64 文件夹内
## 如何卸载?
  将tianWen.dll文件删除即可<br>
  如果出现导致游戏崩溃等问题,卸载即可
## 如何使用翻译功能?
  在 http://api.fanyi.baidu.com/ 上注册翻译api,后把AppID和key填入translator.py内对应位置即可<br>
  不会可以百度<br>
  在游戏内输入 /en 中文字符串 即可将发出的字符串自动转换为英文<br>
  输入 /trans 即可将他人内容翻译为中文,再次输入关闭该功能<br>
